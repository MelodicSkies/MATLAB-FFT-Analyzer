--------------------------------------------------------------- FFT Analyzer ---------------------------------------------------------------

Designed as a side project for audiology lab work
No updates planned as of now
FFT equations are written for 16 bit audio files in mind (files with other sample rates will need to be converted according to their dynamic range)
Only supports .wav and .mp3 files 

--------------------------------------------------------------------------------------------------------------------------------------------

Release V 1.0:
--------------
> extracts waveform data from .mp3 and .wav files
> graphs original spectral data as well as individual frequency components retrieved with fft
